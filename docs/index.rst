This is the documentation of pipeline. You can install the package from `Anaconda.org
<https://anaconda.org/OpenSourceEconomics/ pipeline>`_ with

.. code-block:: bash

    $ conda config --add channels conda-forge
    $ conda install -c opensourceeconomics pipeline
