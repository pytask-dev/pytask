# Marks

Built-in marks are available on `pytask.mark`:

- `pytask.mark.persist`
- `pytask.mark.skipif`
- `pytask.mark.skip_ancestor_failed`
- `pytask.mark.skip_unchanged`
- `pytask.mark.skip`
- `pytask.mark.try_first`
- `pytask.mark.try_last`
